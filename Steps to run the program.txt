Open a terminal and navigate to the project location

Initially create a python virtual environment using the below command:
python -m venv pr-venv

Install the required dependencies by running the command:
pip install -r .\requirements.txt

To Run the program use the below commands
$Env:SRC_BASE_DIR = "data/retail_db"
$Env:DB_HOST = "localhost"
$Env:DB_PORT = "5432"
$Env:DB_NAME = "my_db_name"
$Env:DB_USER = "my_db_user"
$Env:DB_PASS = "Password123"

After running the above commands run python app.py
